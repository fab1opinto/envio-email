Projeto de envio de email com wordpress

* Iniciando a composição com o banco de dados
* Usando volumes e scripts de banco de dados
* Começando nossa camada de front-end
* Aplicativo para enfileirar as mensagens
* Configurando um proxy reverso
* Redes, depêndencia e banco de dados
* Fila e workers
* Escalar é preciso
* 12 Factors
* Sobreescrevendo localmente

front end: nginx
banco de dados: postgres: 9.6
app: python 3.6
worker 1: python 3.6
worker 2: python 3.6
worker 3: python 3.6

comandos
docker-compose up -d (-d em daemon)

executar um comando dentro do serviço para listar os banco de dados postgres
docker-compose exec db psql -U postgres -c '\l'

docker-compose down
para parar os serviços

docker-compose exec db psql -U postgres -f /scripts/check.sql

verificar logs
docker-compose logs -f -t
